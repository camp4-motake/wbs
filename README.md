# WBS

## prerequisites

- Docker Client
- Node.js >=20

## setup environment

1. install dependencies

```sh
npm ci
```

2. run auto WordPress setup

```sh
npm run setup
```

## tasks

```sh
# start wp-env
npm run wp-env start

# dev
npm run dev

# build
npm run build

# lint
npm run lint

# format
npm run format
```
